### Penggunaan Separator
Separator "**sep**" dalam python digunakan sebagai pembatas antara output yang dihasilkan.
 Contoh Code :

# Penggunaan separator
w, x, y, z = 10, 15, 20, 25
print(w, x, y, z, sep=',')
print(w, x, y, z, sep='')
print(w, x, y, z, sep=':')
print(w, x, y, z, sep='- - - - -')
